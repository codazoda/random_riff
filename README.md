Random Riff
===========

Randomly generates a guitar riff using standard scales such as the Pentatonic.

Libraries
=========

Random Riff uses the following libraries.

- jQuery 1.10.2
- jTab 1.1.1
- Raphael 2.1.2
- QuickUser.js 1.0.0
